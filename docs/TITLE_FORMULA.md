# Fórmula de Títulos — The Last Shelter

## Regla #1: SIEMPRE incluir el ADN del show
Todo título DEBE tener estos 3 elementos:
1. **Frío/condición extrema** — temperatura, nieve, blizzard, hielo, avalancha
2. **Construcción/acción** — built, rebuilt, dug, builds, restored
3. **Urgencia** — deadline (días), "alone", "before winter", "start to finish"

Si falta alguno de los 3, el título NO es válido para The Last Shelter.

## Límite: máximo 100 caracteres

## Estructura
```
[Sujeto 3ª persona] + [ACCIÓN CAPS] + [qué construye] + [condición extrema] + | + [deadline/escala]
```

## Palabras Poder (UNA en CAPS por título)
- INCREDIBLE, AMAZING, REBUILT, MASSIVE, ENTIRE

## Sujetos (siempre 3ª persona — el narrador cuenta)
- "He Built...", "She Found...", "Man Builds...", "His Cabin Was..."

## Fórmulas

### BUILD — construcción nueva
```
He/Man Built [CAPS] [tipo] Alone in [temp] | [X] Days Before Winter
```
→ "He Built an INCREDIBLE Log Cabin Alone in -40°F | 42 Days Before Winter"

### UNDERGROUND — excavación/subterráneo
```
He Dug [tipo subterráneo] to Survive [temp] | [X] Days Alone
```
→ "He Dug an Underground Shelter to Survive -40°F | 45 Days Alone"

### RESTORE — restauración de algo abandonado
```
She/He Found [estado] in [lugar frío] and [CAPS] It [condición]
```
→ "She Found an Abandoned Cabin at -30°F and REBUILT It Alone"

### FULL BUILD — gran escala
```
Man Builds [CAPS] [tipo] in [lugar frío] Through [condición extrema]
```
→ "Man Builds AMAZING Off-Grid Cabin in Alaska Through Snow and Ice"

### RESCUE — reconstrucción tras desastre
```
His/Her [qué] Was [destrucción natural] | He/She Rebuilt It in [X] Days
```
→ "His Cabin Was Destroyed by an Avalanche | He Rebuilt It in 60 Days"

## Checklist antes de aceptar un título
- [ ] ¿Tiene elemento frío/extremo? (temperatura, nieve, hielo, avalancha)
- [ ] ¿Tiene acción de construcción?
- [ ] ¿Tiene urgencia o escala?
- [ ] ¿Es 3ª persona?
- [ ] ¿Menos de 100 caracteres?
- [ ] ¿Genera UNA imagen mental clara en 2 segundos?
